About the Purdue Model

The Purdue model is a reference architecture for ICS security that defines 6 levels (0-5). Understanding traffic patterns and security controls at each level is essential for OT defenders. This lab simulates devices and traffic at each level so you can practice detection and response across the full ICS kill chain.
